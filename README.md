# Crear base de datos
Configurar los datos de conexión en config/config.json

# Instalar project
```
npm install
```

# Instalar base de datos
```
npm run installDb
```

# Ejecutar servidor
```
npm start
```

# Postman
En el archivo: ServersMeli.postman_collection.json se encuentra una colección de Postman para llamar a los endpoints
